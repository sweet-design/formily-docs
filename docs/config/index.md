## 什么是 formily？

formily 基于**ant-design-vue**组件库，将此组件库作为设计器，基于此配置进行多端生成器。

## 特性

- 📦 Out of the box, focus on the form configuration layer
- 📋 Perfect documentation system to help developers improve efficiency
- 🏷 TypeScript type definition and implementation
- 🎨 Visualize the form JSON data
- 📱 Support mobile component configuration, flexible extension of third-party libraries
